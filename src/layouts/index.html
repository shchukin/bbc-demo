Описание
Курс посвящен подробному изучению анимации походки и пайплайнам работы с персонажной анимацией
Задача курса — научить человека с любым уровнем подготовки анимировать цикл походки персонажа. Курс полезен для тех, кто делает первые шаги в три-дэ анимации. В качестве примера взят персонаж из мультфильма Sprite Fright (2022) от Blender Studio
Для самостоятельной практики в комплекте с курсом идет blend файл с моделью персонажа, с настроенным ригом Cloud Rig, и подготовленной сценой
